Removing and Downloading a Project Again
========================================

Overview
--------

Removing and downloading a project again is often the quickest way to resolve
local problems.

.. warning::

   Always synchronise your changes before removing a project.

Steps
-----

#. Open the **Projects** list.

#. Locate the project.

#. Open the project menu.

.. figure:: images/remove/menu.png

   Placeholder – Project menu.

#. Tap **Remove from device**.

#. Confirm the action.

#. Download the project again.

.. figure:: images/remove/redownload.png

   Placeholder – Download project again.

Result
------

A fresh copy of the project is stored on the device.
