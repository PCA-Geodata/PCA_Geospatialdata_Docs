Basic Editing
=============

Overview
--------

This guide covers the most common editing tasks.

Adding a point
--------------

#. Tap **Add**.
#. Choose the required point layer.

.. figure:: images/editing/add_point.png

   Placeholder – Add Point.

#. Tap the map.

#. Complete the form.

#. Tap **Save**.

Adding a line
-------------

#. Tap **Add**.
#. Choose a line layer.

.. figure:: images/editing/add_line.png

   Placeholder – Add Line.

#. Tap to create each vertex.

#. Double tap (or use Finish) to complete the line.

#. Complete the form.

#. Tap **Save**.

Editing an existing feature
---------------------------

#. Tap the feature.

#. Select **Edit**.

.. figure:: images/editing/edit_feature.png

   Placeholder – Edit feature.

#. Update the geometry or attributes.

#. Save the changes.
