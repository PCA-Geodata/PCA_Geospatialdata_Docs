Using Map Themes
================

Overview
--------

Map Themes allow you to switch quickly between predefined layer combinations.

Steps
-----

#. Open the required project.

#. Tap the **Map Themes** button.

.. figure:: images/themes/theme_button.png

   Placeholder – Map Themes button.

#. Select the required theme.

.. figure:: images/themes/theme_list.png

   Placeholder – Theme list.

#. The map refreshes automatically.

Examples
--------

* Archaeology
* Geophysics
* Topographic Survey

.. tip::

   Map Themes only change what is displayed. They do not modify any data.
